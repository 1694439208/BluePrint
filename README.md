# BluePrint
可视化节点编辑器

> 基于[CPF](http://cpf.cskin.net/) 框架开发的一款类似于蓝图的组件

> 相比大多数开源的可视化蓝图组件，此控件及其完善，可以开箱即用，架构灵活支持任意的项目需求来自由定制

> 支持实时预览执行节点，也支持动态生成代码，也可以做脑图之类的工具，自由定制，欢迎使用
